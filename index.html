Bem-Vindo

Desenvolvimento

Autossuficiência

Neste curso aprendemos a como trabalhar e desenvolver as nossas competências sozinhos.

Comunicação

Desenvolvemos projetos em grupo para melhorar as nossas habilidades de trabalho em conjunto para determinados fins.

Apoio

Apoio dos professores para o nosso desenvolviento neste curso o qual contribui para uma boa aprendizagem

Opções

Se seguires este curso terás uma carreira com muitas opções de trabalho e também muitas opurtunidades, visto que é uma aréa continuamente crescente.

Aqui terás uma visão geral do que poderás aprender

Competências

As competências que poderás adquirir neste curso

Manutenção

Aqui vais aprender como montar e fazer manutenção de computadores, poderás resolver problemas que tenhas no teu computador pessoal ou abrir uma loja de conserto.

Código

Aprenderás a fazer código e programação, vais entender melhor com tudo funciona por debaixo dos panos e como o teu computador funciona.

Rede

Como construir, montar e manter uma rede, criar grandes redes e entender como os computadores são ligados.

Projetos

Depois de adquirir os conhecimentos, poderás criar os teus próprios projetos independentes, seja por hobbie ou até para entenderes melhor como funciona.
